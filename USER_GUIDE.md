## 简介

OpenMTP 是一款专为 macOS 用户设计的 Android 文件传输应用，提供了比官方 Android File Transfer 更强大、更稳定的文件传输体验。

主要特点：

- 支持传输大于 4GB 的文件
- 支持同时连接多个 Android 设备
- 无线传输模式
- 批量文件操作
- 文件同步功能
- 文件搜索与过滤
- 文件预览功能
- 多标签页支持
- 自定义主题
- 端到端加密传输
- 跨设备剪贴板共享
- 在 Mac 上显示 Android 通知
- 安装/卸载 APK
- 可视化存储空间使用情况
- 大文件/重复文件检测

## 系统要求

- **macOS**：11.0 (Big Sur) 或更高版本
- **Android 设备**：支持 MTP 协议的设备
- **USB 连接**：USB 2.0 或更高版本

## 安装指南

### 方法 1：从官网下载

1. 访问 <https://openmtp.ganeshrvel.com>
2. 根据您的 Mac 芯片类型选择下载版本：
   - Mac Apple Silicon (M1/M2 等)
   - Mac Intel Silicon
3. 下载完成后，打开 DMG 文件
4. 将 OpenMTP 拖放到 Applications 文件夹
5. 首次打开时，可能需要在系统偏好设置中允许应用运行

### 方法 2：使用 Homebrew

```bash
# 安装
brew install openmtp --cask

# 旧版本命令
brew cask install openmtp
```

## 首次使用

1. **连接设备**：使用 USB 数据线将 Android 设备连接到 Mac
2. **设备设置**：在 Android 设备上，当提示 "USB 用于" 时，选择 "文件传输 (MTP)" 模式
3. **授权访问**：在 Android 设备上，允许 "允许访问此设备数据" 的请求
4. **应用启动**：打开 OpenMTP 应用
5. **设备检测**：应用会自动检测并显示连接的设备

## 基本操作

### 文件浏览

- **本地文件**：左侧面板显示 Mac 本地文件系统
- **设备文件**：右侧面板显示 Android 设备文件系统
- **切换存储**：在设备面板顶部，可选择 "内部存储" 或 "SD 卡"
- **视图切换**：可在列表视图和网格视图之间切换
- **排序**：可按名称、大小、日期或类型排序

### 文件传输

- **拖放传输**：直接将文件或文件夹拖放到目标位置
- **复制粘贴**：选择文件后，使用快捷键或右键菜单进行复制粘贴
- **传输队列**：点击底部的传输队列图标查看传输进度
- **暂停/恢复**：在传输队列中可暂停或恢复传输任务

### 文件操作

- **新建文件夹**：点击工具栏的 "新建文件夹" 按钮
- **重命名**：选择文件后，点击右键菜单中的 "重命名"
- **删除**：选择文件后，按 Delete 键或点击右键菜单中的 "删除"
- **批量操作**：按住 Command 键选择多个文件，然后进行批量操作

## 高级功能

### 多标签页

- **新建标签页**：点击标签栏右侧的 "+" 按钮
- **切换标签页**：点击标签页或使用快捷键 Command+数字
- **关闭标签页**：点击标签页上的 "×" 按钮

### 文件收藏/书签

- **添加书签**：在文件浏览器中，右键点击文件夹，选择 "添加到书签"
- **访问书签**：在左侧面板的 "书签" 部分查看已添加的书签
- **管理书签**：点击书签栏右侧的 "管理" 按钮

### 备份功能

- **打开备份管理器**：点击顶部菜单的 "备份" 按钮
- **创建备份**：选择要备份的内容，点击 "开始备份"
- **查看备份历史**：在备份管理器中查看历史备份记录
- **恢复备份**：选择备份记录，点击 "恢复"

### 无线传输

- **打开无线管理器**：点击顶部菜单的 "无线" 按钮
- **扫描设备**：点击 "开始扫描" 查找附近的 Android 设备
- **连接设备**：选择扫描到的设备，点击 "连接"
- **断开连接**：点击 "断开连接" 按钮

### 应用管理

- **打开 APK 管理器**：点击顶部菜单的 "应用" 按钮
- **安装 APK**：点击 "安装 APK" 按钮，选择 APK 文件
- **卸载应用**：在已安装应用列表中，选择应用，点击 "卸载"

### 存储分析

- **打开存储分析器**：点击顶部菜单的 "存储" 按钮
- **查看存储使用情况**：查看总存储、已使用和可用空间
- **查看分区信息**：查看内部存储和 SD 卡的使用情况
- **查看文件类型分布**：查看不同类型文件的存储占用

### 文件分析

- **打开文件分析器**：点击顶部菜单的 "分析" 按钮
- **查看大文件**：在 "大文件" 标签页查看大于指定大小的文件
- **调整大小阈值**：使用滑块调整大文件的最小大小
- **查看重复文件**：在 "重复文件" 标签页查看重复的文件

### 通知镜像

- **启用通知镜像**：在设置中，打开 "通知镜像" 选项
- **查看通知**：在应用中查看 Android 设备的通知
- **管理通知**：点击通知列表中的 "清除全部" 按钮

### 剪贴板同步

- **启用剪贴板同步**：在设置中，打开 "剪贴板同步" 选项
- **使用同步**：在 Mac 上复制内容，会自动同步到 Android 设备，反之亦然

## 故障排除

### 设备未被识别

1. **检查 USB 连接**：确保 USB 数据线连接正常
2. **重启设备**：尝试重启 Android 设备
3. **更换 USB 端口**：尝试使用不同的 USB 端口
4. **检查 USB 模式**：确保设备已设置为 "文件传输 (MTP)" 模式
5. **更新驱动**：确保 Mac 上的 USB 驱动已更新
6. **检查权限**：在设备上允许 "允许访问此设备数据" 的请求

### 传输失败

1. **检查文件大小**：确保文件大小不超过设备存储空间
2. **检查文件权限**：确保有足够的权限访问和传输文件
3. **检查连接**：确保 USB 连接稳定，没有断开
4. **重启应用**：尝试重启 OpenMTP 应用
5. **检查设备存储空间**：确保设备有足够的存储空间

### 应用崩溃

1. **更新应用**：确保使用最新版本的 OpenMTP
2. **检查系统版本**：确保 macOS 版本符合要求
3. **查看日志**：在应用中查看错误日志
4. **重新安装**：尝试卸载并重新安装应用

## 常见问题

### Q: OpenMTP 支持哪些操作系统？

A: OpenMTP 支持 macOS、Windows 和 Linux 操作系统。

### Q: 如何传输大于 4GB 的文件？

A: OpenMTP 原生支持传输大于 4GB 的文件，无需特殊设置。

### Q: 如何同时连接多个 Android 设备？

A: 只需将多个设备通过 USB 连接到 Mac，OpenMTP 会自动检测并显示所有连接的设备，您可以在设备管理器中切换设备。

### Q: 如何使用无线传输模式？

A: 打开无线管理器，点击 "开始扫描" 查找附近的 Android 设备，然后选择设备进行连接。

### Q: 如何备份我的 Android 设备？

A: 打开备份管理器，选择要备份的内容，点击 "开始备份" 即可。

### Q: 如何安装 APK 文件？

A: 打开 APK 管理器，点击 "安装 APK" 按钮，选择要安装的 APK 文件。

### Q: 如何查看设备的存储空间使用情况？

A: 打开存储分析器，查看总存储、已使用和可用空间，以及文件类型分布。

### Q: 如何查找大文件和重复文件？

A: 打开文件分析器，在 "大文件" 标签页查看大于指定大小的文件，在 "重复文件" 标签页查看重复的文件。

### Q: 如何在 Mac 上查看 Android 设备的通知？

A: 在设置中启用 "通知镜像" 选项，然后在应用中查看通知列表。

### Q: 如何同步 Mac 和 Android 设备的剪贴板？

A: 在设置中启用 "剪贴板同步" 选项，然后在 Mac 或 Android 设备上复制内容，会自动同步到另一设备。
